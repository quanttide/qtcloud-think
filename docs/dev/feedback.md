# 用户反馈记录

记录用户在体验过程中提出的问题和改进建议。

---

## 2026-02-19

### 问题 1：AI 追问太多
- **反馈**：用户输入一个想法后，AI 返回了长长的清单问题
- **原因**：prompt 中要求 AI 每次问多个问题
- **修复**：限制每次只问 1 个问题

### 问题 2：拆分用户的多条想法
- **反馈**：用户输入了多个想法（社媒、恋爱游戏、工具等），AI 拆分成 N 个逐一提问
- **修复**：prompt 强调把用户的想法作为整体理解

### 问题 3：空输入死循环
- **反馈**：按回车但没输入内容时，程序进入死循环
- **修复**：处理空输入，直接结束澄清

### 问题 4：长文本输入不友好
- **反馈**：标准 input() 不适合输入长文字
- **尝试**：先考虑 vim 编辑器、Ctrl+D、连续空行等方案
- **修复**：连续两个空行结束输入

### 问题 5：AI 充当裁判判断想法是否清晰
- **反馈**：AI 不应该判断想法"清晰度"，这是用户的权力
- **修复**：改为"思维镜子"模式，AI 只复述不评判

### 问题 6：总结包含 AI 的提问
- **反馈**：生成的结果里包含了 AI 的追问内容
- **修复**：总结 prompt 改为只提取用户想法

### 问题 7：流程太机械
- **反馈**：不应该机械地停止，要根据情况追问
- **修复**：prompt 改为"碰撞"想法而非盘问细节

### 问题 8：对话太封闭
- **反馈**：对话没有服务于创造性认知过程
- **修复**：prompt 改为"思考催化剂"模式，用新角度/隐喻翻译想法
