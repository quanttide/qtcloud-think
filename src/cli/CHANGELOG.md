### Added

- Repo layer: worlds / periods / domains query with journal storage
- Intention queries: by world/period/domain, all with filters, by UUID
- Analyze: `track_evolution()` for domain evolution tracing
- Config: journal path from env vars
- Dependencies: quanttide-think 0.1.0, quanttide-agent 0.1.0
