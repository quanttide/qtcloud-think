# Changelog

All notable changes to qtcloud-think Studio will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [studio/0.0.1-alpha.1] - 2026-04-25

### Added
- 认知棱镜交互式阅读器：双栏布局，日志 + 认知卡片
- 操作指引：每个认知模式提供可执行的下一步行动
- Linux 桌面支持

### Changed
- Flutter SDK 升级到 3.11.0
- flutter_lints 升级到 6.0.0
- 模块重组：models/journal + samples/journal
