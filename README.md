# 量潮思考云 (`qtcloud-think`)

思维外脑 - 思维收集与澄清工具

## 什么是思维外脑？

当人类处于默认神经网络状态时，帮助人类承载需要切换到任务正向网络的任务，使人类可以专注于放松。

核心功能：收集人类思维流，澄清想法念头，记录清晰的知识结构。
